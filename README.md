# Enigma

In order to run this program in development, perform the following commands:

1 .  Clone the repository

```
$ git clone
```

2 . cd into the directory
```
$ cd enigma
```

3 . Run the program
```
$ ruby enigma.rb [input one] [input two]
```

4 . Run the tests
```
$ ruby test/encryptor_test.rb
```
